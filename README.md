# 回魂夜 · 纸人招魂

一款中式规则怪谈风格的恐怖文字冒险游戏。

末班车把你带回爷爷所在的纸人村。今夜，你得守着三条死规矩，活到天亮。

> 浏览器打开即玩；手机上用「添加到主屏幕」可全屏启动，像一个 App。

## 玩法
- 纯文字交互 + 场景探索，靠你的选择推进剧情
- 多个章节、多种结局与死法，藏着不少分支和隐藏线索
- 有「血量 / 理智 / 恐惧」三项状态，关系到你能不能撑到最后
- 支持本地存档，随时存随时读

## 本地运行
游戏是纯静态网页，用任意静态服务器指向本目录即可，例如：

```
npx serve .
```

（直接双击 `index.html` 会因浏览器的模块安全策略打不开，需要用「服务器」方式访问。）

## 部署
所有文件都是静态资源，可托管在 GitHub Pages、Cloudflare Pages、对象存储等任意静态网站服务上。
GitHub Pages：在仓库 Settings → Pages 里，把来源设为 `main` 分支根目录即可。
