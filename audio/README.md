# 背景音乐 / 音效文件放这里

把无版权(CC0 / 可商用)mp3 放到本目录,**用下列固定文件名**,游戏会自动改用真实音频;
文件不存在时自动回退到程序合成版(不会报错)。改动后记得 bump 缓存令牌 + 部署。

| 文件名 | 用途 | 触发时机 | 建议 |
|---|---|---|---|
| `bgm_ambient_loop.mp3` | 全程氛围铺底 | 第一~七章常态 | 可无缝循环, 低沉阴森, < 1.5MB |
| `bgm_suona_climax.mp3` | 唢呐送葬悲调 | 第八章冥婚 / 第九章阴阳路 / 第十章回魂 | 唢呐为主, 凄厉, 可循环 |
| `bgm_ending_relief.mp3` | 哀婉释怀 | 结局节点(真名/回魂等) | 二胡/笛, 释怀收束 |
| `sfx_jumpscare.mp3` | 惊吓瞬间 | jumpscare(选填) | 2~5 秒, 闷锣+唢呐破音 |

## 授权
只放 **CC0 或明确可商用** 的素材, 逐曲核对许可。本目录文件会随前端一起部署到公网, 等同商用。

## 接入位置
逻辑在 `vue-frontend/js/audio.js`(BGM_FILES 配置 + startBgm 文件优先/合成兜底)。
按章节切换在 `vue-frontend/js/app.js` 的 `_updateBgm`。
